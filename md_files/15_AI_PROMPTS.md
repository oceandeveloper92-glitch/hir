# 🤖 AI PROMPTS — Best Prompts For Every Situation
<!-- Copy-paste prompts for common tasks -->

---

## 🚀 Project Start
```
New project: [name]
Tech: [Laravel/WordPress/Next.js/etc.]
Goal: [What the project does]

Copy templates from "E:\Wordk-- Desktop\=md files All\" to this project folder.
Fill PROJECT_CONTEXT.md with all project details.
Set up folder structure.
After every change, update project_status.md with date/time (IST)/file/what/why.
```

## 📂 Session Start (EVERY TIME)
```
Read PROJECT_CONTEXT.md, project_status.md, and TODO.md first.
Tell me where we left off and what's next.
```

## 🐛 Bug Fix
```
Bug: [describe the bug — what happens vs what should happen]
File: [which file/page if you know]

1. [step 1]
2. [step 2]

1. Identify root cause
2. Explain what's wrong
3. Fix with minimal changes
4. Update BUGS.md
```
