# Lessons Learned (Global Log)

### [2026-04-29] GardenAsia: Dynamic DOM & Event Delegation
- **Problem**: Shopify variant pickers refresh the DOM via AJAX, which wipes out standard JS event listeners.
- **Solution**: Always use **Document-level Event Delegation** (e.g., `document.addEventListener('click', ...)` with selector checks) for elements inside dynamic sections. This ensures persistent functionality without needing to re-initialize scripts.
