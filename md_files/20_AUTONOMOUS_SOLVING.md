# 🤖 Autonomous Problem Solving

## Objective
To build systems that are self-sufficient, robust, and capable of identifying/fixing their own issues with minimal human intervention.

## AI Instructions
1. **Self-Healing Code**: Always include defensive error handling and clear logging.
2. **Proactive Alerts**: Design systems that notify the USER (or the AI) before a major failure occurs.
3. **Independent Research**: If a technical blocker arises, the AI must use all available tools (web search, documentation) to find a solution autonomously before asking for help.
4. **Resilience**: Prioritize stable, long-term technical solutions over "quick fixes".
